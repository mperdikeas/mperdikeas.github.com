1. connect to various pods

    2. One can always use the k9s program (set up and used as advised here). Alternatively it is possible to directly open a shell into, or transfer files from a pod using the following incantations from one's own shell:

    • to retrieve pod names
      • kubectl get pods --context=cognitor-aks --namespace=cognitera-platform
    • to open a shell into a pod (identified by name)
      • kubectl exec -it --context=cognitor-aks --namespace=<namespace> pod/<pod name> -- bash
    • retrieve from or transfer files to a pod
      • kubectl cp --context=cognitor-aks --namespace=<namespace> ./local-file.txt <pod name>:/tmp/remote-file.txt
    • setup port forwarding with local machine
      • kubectl port-forward --context=hetzner-staging --namespace=cognitera-platform svc/postgresql 35001:5432
