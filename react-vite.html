1. Vite stalling in dev server

    2. In eary 2024 I began experiencing the dreaded vite development server's stalling. After much frustration I encountered this SO question which pointed to this vite documentation page.

    Following the instructions on said page I only, I added the line: * - nofile 65536 to the file /etc/security/limits.conf and restarted. Vite development server has been working flawlessly ever since.
