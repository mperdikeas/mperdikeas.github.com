• PDS
    • The Planetary Data System format is a NASA specification for creating products and basically associating observation files containing telemetry with a structured description of their content.
    Below find some useful links:
    • landing page
    • Concepts document
    • Data Provider's Handbook
    • Standards Reference
    • PDS4 XSD schema
    • PDS4 data dictionary (apparently produced automatically from the schema above)
    • A simple, sample product in the form of a 1D array (and its associated label)

    Also, the above reference documents and the example are available in the RAWDAR repository.
