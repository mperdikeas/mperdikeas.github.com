1. how to install Ruby 2.0.0 on Ubuntu 12.04
    2. Instructions taken from here.
    Basically, I followed all the steps, indicated above, only invoking sudo and fixing a switch in the tar invocation:
    1. sudo apt-get -y update
    2. sudo apt-get -y install build-essential zlib1g-dev libssl-dev libreadline6-dev libyaml-dev
    3. cd /tmp
    4. wget http://cache.ruby-lang.org/pub/ruby/2.0/ruby-2.0.0-p247.tar.gz
    5. sudo tar xvfz ruby-2.0.0-p247.tar.gz
    6. cd ruby-2.0.0-p247/
    7. sudo ./configure --prefix=/usr/local
    8. sudo make
    9. sudo make install
    Some steps (e.g. 1 and 2 I think) complained a bit but in the end ruby -v produces "ruby 2.0.0p247 (2013-06-27 revision 41674) [i686-linux]" and the constant RUBY_VERSION in irb evaluates to "2.0.0".
