1. Configure PrimeFaces file upload component
    2. The default example found in the Prime Faces showcase (as accessed X.2012) will not work out of the box. You have to set up the following first:
    1. declare the Prime Faces upload filter in web.xml as follows:
      2.       
         PrimeFaces FileUpload Filter
         org.primefaces.webapp.filter.FileUploadFilter
      
      
         PrimeFaces FileUpload Filter
         Faces Servlet
    2. Adding two libraries to your application: Apache commons-file-upload and Apache commons-io. So you need to download them and add them to your WEB-INF/lib path:
      1. Here you can download Apache Commons io
      2. Here you can download Apache Commons File Upload
