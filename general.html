1. change hostname in Ubuntu

    simply edit /etc/hostname and /etc/hosts (for 127.0.0.1). To flush the previous hostname from various applications a reboot may be required.

  2. setting HTTP proxy in Ubuntu

    export http_proxy="http://foo.bar:8080"
    subsequent wget requests will redirect to foo.bar:8080
  3. package managers

    apt-get is used in Debian, Ubuntu and rpm in Fedora, Redhat, Suse. apt-get is better.

  4. curl

    curl is a Linux utility that can be used to simulate HTTP GET and HTTP POST requests. A simple:

    #curl www.in.gr
    ...will execute an HTTP GET against www.in.gr.
  5. SSE time

    get time in SSE:

    date +%s
