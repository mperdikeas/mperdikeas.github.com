1. how to list all Python modules in a system (built-in and from custom PYTHONPATH)

    2. Launch Python and in the REPL type help('modules'). This will list all modules whether built-in or modules stored in a custom PYTHONPATH. By using help('somemodule') you can obtain more information including the path to the *.py file for each module. From the file information where you can easily see if the module is built-in or rather present in some custom PYTHONPATH.

    3. sudo apt-get install idle3

    4. pip freeze
  4. how to find the Python sys.path (and the root locations of all Python packages)
